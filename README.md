# prefreshybot

## Getting started

### 1. Install dependencies

```bash
bun install
```

### 2. Configure environment variables

```bash
cp .env.example .env
```

### 3. Deploy slash commands

```bash
bun run deploy

# Global
GLOBAL=true bun run deploy
```

### 4. Run the bot

```bash
# Development
bun run dev

# Production
bun run start
```
